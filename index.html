GitHub Gist セットアップ

データを保存・共有するために、GitHub Gistの情報を入力してください。

新規作成の場合は空欄のまま「保存」をクリック
書き込み権限が必要な場合のみ入力

GitHub Personal Access Tokenの取得方法：

  1. GitHubにログイン
  2. Settings → Developer settings → Personal access tokens
  3. 「Generate new token」をクリック
  4. 「gist」にチェックを入れて作成
